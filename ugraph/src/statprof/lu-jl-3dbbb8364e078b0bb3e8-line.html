86 """
87 lufact(A [,pivot=Val{true}]) -> F::LU
88
89 Compute the LU factorization of `A`.
90
91 In most cases, if `A` is a subtype `S` of `AbstractMatrix{T}` with an element
92 type `T` supporting `+`, `-`, `*` and `/`, the return type is `LU{T,S{T}}`. If
93 pivoting is chosen (default) the element type should also support `abs` and
94 `<`.
95
96 The individual components of the factorization `F` can be accessed by indexing:
97
98 | Component | Description |
99 |:----------|:------------------------------------|
100 | `F[:L]` | `L` (lower triangular) part of `LU` |
101 | `F[:U]` | `U` (upper triangular) part of `LU` |
102 | `F[:p]` | (right) permutation `Vector` |
103 | `F[:P]` | (right) permutation `Matrix` |
104
105 The relationship between `F` and `A` is
106
107 `F[:L]*F[:U] == A[F[:p], :]`
108
109 `F` further supports the following functions:
110
111 | Supported function | `LU` | `LU{T,Tridiagonal{T}}` |
112 |:---------------------------------|:-----|:-----------------------|
113 | [`/`](@ref) | ✓ | |
114 | [`\\`](@ref) | ✓ | ✓ |
115 | [`cond`](@ref) | ✓ | |
116 | [`inv`](@ref) | ✓ | ✓ |
117 | [`det`](@ref) | ✓ | ✓ |
118 | [`logdet`](@ref) | ✓ | ✓ |
119 | [`logabsdet`](@ref) | ✓ | ✓ |
120 | [`size`](@ref) | ✓ | ✓ |
121
122 # Example
123
124 ```jldoctest
125 julia> A = [4 3; 6 3]
126 2×2 Array{Int64,2}:
127 4 3
128 6 3
129
130 julia> F = lufact(A)
131 Base.LinAlg.LU{Float64,Array{Float64,2}} with factors L and U:
132 [1.0 0.0; 1.5 1.0]
133 [4.0 3.0; 0.0 -1.5]
134
135 julia> F[:L] * F[:U] == A[F[:p], :]
136 true
137 ```
138 """

275 function det(A::LU{T}) where T
276 n = checksquare(A)
277 A.info > 0 && return zero(T)
278 P = one(T)
279 c = 0
280 @inbounds for i = 1:n
281 P *= A.factors[i,i]
282 if A.ipiv[i] != i
283 c += 1
284 end
285 end
286 s = (isodd(c) ? -one(T) : one(T))
287 return P * s
288 end
