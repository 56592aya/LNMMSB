49 unsafe_wrap(Array, pointer::Ptr{T}, dims, own=false)
50
51 Wrap a Julia `Array` object around the data at the address given by `pointer`,
52 without making a copy. The pointer element type `T` determines the array
53 element type. `dims` is either an integer (for a 1d array) or a tuple of the array dimensions.
54 `own` optionally specifies whether Julia should take ownership of the memory,
