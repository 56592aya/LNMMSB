5 """
6 MvNormalCanon
7
8 Multivariate normal distribution is an [exponential family distribution](http://en.wikipedia.org/wiki/Exponential_family),
9 with two *canonical parameters*: the *potential vector* ``\\mathbf{h}`` and the *precision matrix* ``\\mathbf{J}``.
10 The relation between these parameters and the conventional representation (*i.e.* the one using mean ``\\boldsymbol{\\mu}`` and
11 covariance ``\\boldsymbol{\\Sigma}``) is:
12
13 ```math
14 \\mathbf{h} = \\boldsymbol{\\Sigma}^{-1} \\boldsymbol{\\mu}, \\quad \\text{ and } \\quad \\mathbf{J} = \\boldsymbol{\\Sigma}^{-1}
15 ```
16
17 The canonical parameterization is widely used in Bayesian analysis. We provide a type `MvNormalCanon`,
18 which is also a subtype of `AbstractMvNormal` to represent a multivariate normal distribution using
19 canonical parameters. Particularly, `MvNormalCanon` is defined as:
20
21 ```julia
22 immutable MvNormalCanon{P<:AbstractPDMat,V<:Union{Vector,ZeroVector}} <: AbstractMvNormal
23 μ::V # the mean vector
24 h::V # potential vector, i.e. inv(Σ) * μ
25 J::P # precision matrix, i.e. inv(Σ)
26 end
27 ```
28
29 We also define aliases for common specializations of this parametric type:
30
31 ```julia
32 const FullNormalCanon = MvNormalCanon{PDMat, Vector{Float64}}
33 const DiagNormalCanon = MvNormalCanon{PDiagMat, Vector{Float64}}
34 const IsoNormalCanon = MvNormalCanon{ScalMat, Vector{Float64}}
35
36 const ZeroMeanFullNormalCanon = MvNormalCanon{PDMat, ZeroVector{Float64}}
37 const ZeroMeanDiagNormalCanon = MvNormalCanon{PDiagMat, ZeroVector{Float64}}
38 const ZeroMeanIsoNormalCanon = MvNormalCanon{ScalMat, ZeroVector{Float64}}
39 ```
40
41 A multivariate distribution with canonical parameterization can be constructed using a common constructor `MvNormalCanon` as:
42
43 MvNormalCanon(h, J)
44
45 Construct a multivariate normal distribution with potential vector `h` and precision matrix represented by `J`.
46
47 MvNormalCanon(J)
48
49 Construct a multivariate normal distribution with zero mean (thus zero potential vector) and precision matrix represented by `J`.
50
51 MvNormalCanon(d, J)
52
53 Construct a multivariate normal distribution of dimension `d`, with zero mean and a precision matrix as `J * eye(d)`.
54
55 # Arguments
56 - `d::Int`: dimension of distribution
57 - `h::Vector{T<:Real}`: the potential vector, of type `Vector{T}` with `T<:Real`.
58 - `J`: the representation of the precision matrix, which can be in either of the following forms (`T<:Real`):
59 1. an instance of a subtype of `AbstractPDMat`
60 2. a square matrix of type `Matrix{T}`
61 3. a vector of type `Vector{T}`: indicating a diagonal precision matrix as `diagm(J)`.
62 4. a real number: indicating an isotropic precision matrix as `J * eye(d)`.
63
64 **Note:** `MvNormalCanon` share the same set of methods as `MvNormal`.
65 """
